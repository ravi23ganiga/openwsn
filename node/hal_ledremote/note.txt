>> 自适应LED灯实验
>> ================================================================
>>
>> 演示
>>
>> 取3个node,分别记为ABC
>> 其中,A为无线光敏传感器node,BC为无线被控LED灯
>>
>> 当用手遮挡光敏传感器时,A发出指令数据帧,给BC两个node,通知BC两个node将LED亮度调亮些,当手离开时,LED自动变暗.
>>
>> LED亮暗控制可通过输出PWM波给LED实现.
>> (要看看我们的硬件上是否可行? 如果不能实现ＰＷＭ控制的话，就用ＬＥＤ的数据量来表示亮度)
