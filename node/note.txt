1. 安装版本管理客户端TortoiseSVN
http://tortoisesvn.tigris.org/

2. 建立本地工作副本
创建目录 d:\dev\openwsn\source\tongjiwsn\
该目录必须为空
选择目录d:\dev\openwsn\source\tongjiwsn，Windows资源管理器右键菜单 TortoiseSVN => checkout
选择 https://tongjiwsn.googlecode.com/svn/trunk/tongjiwsn
Checkout 到 d:\dev\openwsn\source\tongjiwsn\

如何从google code获取代码，请查阅此处说明
http://code.google.com/p/tongjiwsn/source/checkout

申请到的google帐号需由管理员加入到该项目成员列表中。

# Project members authenticate over HTTPS to allow committing changes.
svn checkout https://tongjiwsn.googlecode.com/svn/trunk/tongjiwsn --username <你的google帐号>

注意：如果要能上传代码的话，还需要一个Password。Password的获得需要在上页面点击<googlecode password>获得。
但是这个链接必须要用自己的google帐号登录进取才可以看到，且这个密码是google自动为你生成，并非是你google帐号密码。如果不登录的话，只能看到如下链接：

# Non-members may check out a read-only working copy anonymously over HTTP.
svn checkout http://tongjiwsn.googlecode.com/svn/trunk/tongjiwsn

如果仅是只读下载，不需要commit，则无需上述步骤

上述工作完成后，我们就可以在d:\dev\openwsn\source\tongjiwsn\目录中工作，成功的工作直接commit到网络svn服务器中，同时选择update可从网络更新其他人的最新进展到自己的计算机中。



在使用AVR Studio 4.x和Portable WinAVR编译时，需要设置compiler和make的路径
如按照我们的规范安装上述软件，应该设置如下：

D:\portable\WinAVR-20080610\bin\avr-gcc.exe
D:\portable\WinAVR-20080610\utils\bin\make.exe
