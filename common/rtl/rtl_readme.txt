
this folder contains the "Runtime Library"(rtl) used in openwsn project. "rtl" is a general library. you can used it in "hal" or "service" layer.

usually you can use some standard functions provided by your developing environment such as GNU C or ADS. this "rtl" can be regarded as an replacement or extension of it if you cannot access those functions.  

rtl library can be used in hal, kernel and service. 
they're totally hardware independent and fully quanlify ANSI c standard.
most of the data structures are placed in rtl folder.
